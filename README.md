## Install dependencies

./mvnw clean install

## Run the application

./mvnw spring-boot:run

## Run the tests

./mvnw clean test

## Compile the application

./mvnw clean compile

## Run mysql with docker

docker run -p 3307:3306 -e MYSQL_ROOT_PASSWORD=my-secret-pw -d mysql
